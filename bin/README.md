# Executable Scripts

This directory contains all executable scripts for running, testing, and managing the application.

## Scripts

- **setup** - Initial setup and environment configuration
- **start** - Start the complete application stack
- **start-backend** - Start only the backend API server
- **start-frontend** - Start only the frontend development server
- **stop** - Stop all running services
- **test** - Run the test suite
- **demo** - Launch the demo environment
- **test-demo** - Test the demo functionality
- **validate** - Validate the installation and configuration

## Usage

All scripts are executable. Run them directly from anywhere in the project:

```bash
# From project root
./bin/setup
./bin/start

# Or with absolute path
bin/demo
```

## Quick Start

1. Run initial setup: `./bin/setup`
2. Start services: `./bin/start`
3. Run tests: `./bin/test`
4. Try the demo: `./bin/demo`
